# docs/ai/

This directory contains documents intended primarily for AI agents (Claude, Codex, Copilot, etc.) operating on this repository. Human contributors are welcome to read them too, but the primary audience is automation: the agent workflow, project map, decision log, review playbook, issue-intake reference, and environment-specific knowledge for re-running automation safely.

このディレクトリは、本リポジトリ上で動く AI エージェント（Claude / Codex / Copilot 等）が読むことを主目的としたドキュメントを集めた場所です。人間が読んでも問題ありません。エージェント作業手順、プロジェクトマップ、判断ログ、レビュー観点、`evo issue show` 入口仕様、環境依存の再発防止ノウハウが含まれます。

## Contents

- [AGENT_WORKFLOW.md](./AGENT_WORKFLOW.md) — issue から branch / PR / handoff まで進める手順
- [PROJECT_MAP.md](./PROJECT_MAP.md) — repo を開いた agent 向けの「どこから読むか」地図
- [DECISIONS.md](./DECISIONS.md) — 仕様判断の理由ログ
- [REVIEW_PLAYBOOK.md](./REVIEW_PLAYBOOK.md) — 差分レビュー観点
- [issue-intake.md](./issue-intake.md) — `evo issue show` の使い方と読み取り対象
- [knowledge/](./knowledge/) — 環境依存・運用事故の再発防止ノウハウ

Human-facing docs (roadmap, contributing, versioning) live one level up under `docs/`.
