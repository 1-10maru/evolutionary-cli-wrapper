# Knowledge

ここは、運用中に見つかった再発防止ノウハウを残す場所です。

## 何を置くか

- 環境依存のハマりどころ
- shell / terminal / shim の事故メモ
- 一度解いたら次も役立つ確認手順
- 仕様ではなく、運用や調査で得た知見

## 何を置かないか

- 長期の仕様判断
  - `docs/ai/DECISIONS.md` に置く
- 初心者向けのセットアップ本文
  - `README.md`（バイリンガル、日本語クイックインストールセクションを含む）を使う

## 命名ルール

- 環境単位で分ける
  - `windows-shells.md`
  - `zellij-windows.md`
- 症状より、あとで探しやすい単位名を優先する

## 追加ルール

- 再現した環境事故は、修正後にここへ残す
- docs は UTF-8 で編集する
