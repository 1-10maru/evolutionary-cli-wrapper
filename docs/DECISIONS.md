# Decisions

このファイルは、複数人や複数 AI エージェントで進める時に「なぜこうなっているか」を短く残すためのメモです。

## 2026-04-09

### UI は通常 1 行

- Evo のリアルタイム表示は通常 1 行に固定する
- 進化、強い rescue、大きい chance の時だけ 2-3 行まで許可する
- 理由: 目立たせたいが、Claude / Codex の本文を邪魔しすぎないため

### 育成モンスターは PC 全体で 1 体

- EvoPet は project ごとではなく PC 全体で 1 体育てる
- 理由: プロジェクト横断で育成の連続性を感じられる方がゲーム性が高いため

### 学習データは project-local、マスコットは global

- `.evo` は project-local に保存する
- EvoPet の状態は `EVO_HOME/.evo` に保存する
- 理由: プロジェクトごとの知識と、横断的なゲーム進行を分離するため
